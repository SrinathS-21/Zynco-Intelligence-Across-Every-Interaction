# Mail Agent

This is a standalone Next.js Mail Agent app with login and Gmail classifier features.

Essential structure:

- `src/` application code (routes, API handlers, UI, hooks, services)
- `public/` static assets
- `prisma/` Prisma schema
- `package.json` project scripts and dependencies
- `next.config.ts`, `postcss.config.mjs`, `tsconfig.json` build/config files

Run locally:

```bash
npm install
npm run dev
```
